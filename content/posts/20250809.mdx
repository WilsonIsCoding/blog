## 題解

```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfTwo = function (n) {
    while (n > 1) {
        n = n / 2
        if (!Number.isInteger(n)) return false
    }
    return n == 1
};
```

時間複雜度：O(logn)
空間複雜度：O(1)
